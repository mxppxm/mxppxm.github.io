🤖 记账助手 Skill：给你的 AI 搭档装个「财务大脑」

一款 AI 记账技能——说人话就能记账，自动归类、月度统计、预算告警一条龙。

这是什么？

一个 AI 技能，让你的 AI 搭档学会管钱。

和它说话就像跟朋友聊天一样自然：

你 → 今天支出30元买咖啡
AI → ✅ 已记录：支出 30.00元 – 餐饮

📊 2026-06 月度小结：
   收入 5000.00元  |  支出 130.00元  |  结余 4870.00元

💡 提醒：本月支出已达预算的 72%，还剩 170 元

核心功能

📝 自然语言记账

不需要打开 App、不需要选择分类，直接说就行：

你说的话 自动识别结果
支出30元买咖啡 餐饮 · 30元
工资到账5000 工资 · 5000元
昨天打车花了50元 交通 · 50元（日期自动为昨天）
花了100元吃饭 餐饮 · 100元
收入5000元工资 工资 · 5000元

📅 日期智能解析

支持多种日期表达，不写默认今天：

昨天       → 自动计算
前天       → 自动计算
3月5号     → 当年当日
5天前      → 自动计算
2024-01-15 → 标准日期

🏷️ 自动归类

内置 7 类支出 + 4 类收入的关键词映射，智能匹配：

支出类别 自动匹配关键词
餐饮 吃、饭、咖啡、奶茶、外卖、火锅、零食…
交通 打车、地铁、公交、滴滴、加油、停车…
购物 衣服、淘宝、京东、超市、日用…
娱乐 电影、游戏、KTV、旅游、会员…
医疗 看病、药、医院、体检…
学习 书、课程、培训、教材…
居住 房租、水电、物业、网费…
收入类别 自动匹配关键词
工资 工资、薪水、月薪
奖金 奖金、年终、绩效
兼职 兼职、接单、外包
投资 理财、基金、股票、利息

💰 预算管理

预算 4000元        → 设置本月预算
预算 5000元 下个月 → 提前设置下月预算（到月自动切换）

三级告警机制： - 💡 支出达预算 70% → 温和提醒 - ⚠️ 支出达预算 90% → 告警 - 🚨 超支 → 严重告警

📊 月度统计

每次记账自动输出当月汇总：总收入、总支出、结余、预算剩余。

技术架构

用户消息 → parse_message()
           ├── 关键词加权 → 判断收支类型
           ├── 提取金额（Decimal 精度防浮点误差）
           ├── parse_date() → 智能日期解析
           └── 关键词匹配 → 自动归类

记账本（CSV） ← add_entry()
     ↓
月度统计 → 收入/支出/结余
     ↓
预算检查 → 70% / 90% / 超支三级告警
     ↓
结构化响应 → handle() 返回 dict

为什么用 CSV？ 人类可读——Excel 能打开、cat 能看、grep 能搜，个人记账足够。

快速开始

下载源码

git clone https://github.com/mxppxm/ai-skills.git
cd ai-skills/记账助手

安装

将 记账助手.md 和 记账助手.py 放入 AI 平台的技能目录，重启即可生效。

开记

直接对你的 AI 搭档说人话：

支出30元买咖啡 工资到账5000 预算 4000元

文件结构

记账助手/
├── 记账助手.md     # Skill 定义（描述、触发词、使用说明）
├── 记账助手.py     # Python 实现（消息解析、CSV 读写、统计预算）
├── 记账本.csv      # 自动生成，所有账目记录
└── 预算设置.json   # 自动生成，预算配置

技术栈

  • 语言: Python 3（仅标准库，零第三方依赖）
  • 存储: CSV（账本）+ JSON（预算配置）
  • 精度: decimal.Decimal 防浮点误差
